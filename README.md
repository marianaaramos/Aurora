# Aurora
Página web desenvolvida para estudo e aprimoramento de propiedades básicas de desenvolvimento frontend. Suas funcionalidades são baseadas na ideia inicial, desenlvolvida durante uma das disciplinas de Análise e Desenvilvimento de Sistemas, de desenvolver um aplicativo integrado a um acessório (anel, pulseira ou colar) contendo um rastreador acionável em caso de ameaça física, acionando ajuda imediatamente. Iniciando meus aprendizados sobre frontend, Aurora foi desenvolvido em formato de página web.

# Tecnologias
- HTML
- CSS
- JavaScript

# Funcionalidades
- Página inicial intuitiva, contendo um menu simples e responsivo;
- Botões interativos, tanto visual quanto funcionalmente, utilizando CSS e JavaScript;
- Formulários com interações visuais em JavaScript, mensagens, alertas e redirecionamentos;
- Imagens que possuem efeitos de transição ao movimento do cursor.
